# Обзор продукта

SENTINEL — комплексная платформа AI-безопасности для защиты от атак на LLM (prompt injection, jailbreaks, PII-утечки) и тестирования AI-систем (red team).

## Основные возможности

- **BRAIN (Детекция)**: 258 движков на основе ML + правил, странная математика™
- **SHIELD (Защита)**: DMZ на чистом C, латентность <1ms, 20 протоколов
- **STRIKE (Атака)**: Red team платформа, 39K+ payloads, AI-планировщик атак
- **FRAMEWORK (SDK)**: Python SDK, pip install, FastAPI интеграция
- **IMMUNE (EDR)**: Endpoint-защита на уровне ядра для DragonFlyBSD/FreeBSD

## Целевые сценарии

- Защита production LLM-сервисов от prompt injection
- Red team тестирование AI-систем перед релизом
- Compliance-интеграция (SOC 2, HIPAA, PCI-DSS)
- Исследование безопасности AI для академических/корпоративных целей

## Ценностное предложение

Единственная open-source платформа, объединяющая защиту + атаку + SDK в одном решении. "Strange Math" — уникальные алгоритмы детекции (дрифт-анализ, энтропия, геометрические паттерны), недоступные в коммерческих решениях.
